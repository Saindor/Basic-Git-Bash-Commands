Список основных команд для работы с Git Bash!





1.pwd – показать, где ты сейчас находишься (путь к папке)



2.ls – показать файлы и папки в текущей директории



3.cd имя\_папки – перейти в другую папку



4.cd .. – подняться на уровень выше



5.mkdir имя\_папки – создать новую папку



6.touch имя\_файла – создать новый файл



7.rm имя\_файла – удалить файл



8.rm -r имя\_папки – удалить папку с содержимым



9.git init – создать новый репозиторий



10.git clone ссылка – скопировать репозиторий с GitHub



11.git status – показать текущее состояние проекта (изменения, новые файлы)



12.git add имя\_файла – добавить файл к коммиту



13.git add . – добавить все файлы



14.git commit -m "текст" – сохранить изменения с сообщением



15.git push – отправить изменения на GitHub



16.git pull – скачать обновления с GitHub



17.git log – показать историю коммитов



18.git branch – показать все ветки



19.git checkout имя\_ветки – перейти в другую ветку



20.git checkout -b имя\_ветки – создать и перейти в новую ветку



21.git remote -v – показать подключённый удалённый репозиторий



22.clear – очистить экран



23.exit – выйти из Git Bash
